# Kubernetes 部署方式

Kubernetes 集群有多种部署方式，本文档介绍常用的部署方法和工具。

## 目录结构

- [kubeadm](./kubeadm/)
- [Kubespray](./kubespray/)

## 部署方式概述

### kubeadm
kubeadm 是 Kubernetes 官方提供的集群部署工具，适合快速搭建和测试环境。

### Kubespray
Kubespray 是基于 Ansible 的 Kubernetes 集群部署工具，适合生产环境的大规模部署。
