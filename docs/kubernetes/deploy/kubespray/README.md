# Kubespray 部署

Kubespray 是基于 Ansible 的 Kubernetes 集群部署工具，适合生产环境的大规模部署。

## 特点

- 基于 Ansible，易于自动化
- 支持多种操作系统
- 支持高可用集群部署
- 丰富的配置选项
- 适合生产环境

## 基本部署流程

1. 准备 Ansible 环境
2. 克隆 Kubespray 项目
3. 配置 inventory 文件
4. 自定义集群配置
5. 执行部署 playbook
