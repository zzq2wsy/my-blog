# Linux 基础常用命令

本文档记录了 Linux 系统中最常用的命令，包括文件操作、文本处理、系统管理等。

## 文件和目录操作

### 基本命令

- `ls` - 列出目录内容
- `cd` - 切换目录
- `pwd` - 显示当前工作目录
- `mkdir` - 创建目录
- `rmdir` - 删除空目录
- `rm` - 删除文件或目录
- `cp` - 复制文件或目录
- `mv` - 移动/重命名文件或目录
- `touch` - 创建空文件或更新文件时间戳

### 文件查看

- `cat` - 查看文件内容
- `more` - 分页查看文件内容
- `less` - 分页查看文件内容（可向前翻页）
- `head` - 查看文件前几行
- `tail` - 查看文件后几行

### 文件查找

- `find` - 查找文件
- `locate` - 快速查找文件
- `grep` - 文本搜索
- `which` - 查找命令位置
- `whereis` - 查找命令位置及相关文件

## 文本处理

- `echo` - 输出文本
- `printf` - 格式化输出
- `sed` - 流编辑器
- `awk` - 文本处理工具
- `sort` - 排序
- `uniq` - 去重
- `wc` - 统计行数、单词数、字符数

## 系统信息

- `uname` - 显示系统信息
- `whoami` - 显示当前用户
- `id` - 显示用户ID和组ID
- `date` - 显示/设置日期时间
- `uptime` - 显示系统运行时间
- `df` - 显示磁盘使用情况
- `du` - 显示目录或文件大小
- `free` - 显示内存使用情况

## 进程管理

- `ps` - 查看进程
- `top` - 实时显示进程状态
- `kill` - 终止进程
- `killall` - 按名称终止进程
- `jobs` - 查看后台任务
- `bg` - 将任务放到后台
- `fg` - 将任务调到前台
- `nohup` - 忽略挂起信号运行命令

## 网络命令

- `ping` - 测试网络连接
- `ifconfig` - 配置网络接口
- `ip` - 显示/操作路由、网络设备等
- `netstat` - 显示网络状态
- `ss` - 显示套接字统计信息
- `wget` - 下载文件
- `curl` - 传输数据

## 权限管理

- `chmod` - 修改文件权限
- `chown` - 修改文件所有者
- `chgrp` - 修改文件所属组
- `sudo` - 以超级用户身份执行命令

## 压缩解压

- `tar` - 打包/解包
- `gzip` - 压缩/解压
- `zip` - 压缩/解压
- `unzip` - 解压zip文件
