# Debian系 Linux

Debian系 Linux 是基于 Debian 的 Linux 发行版，使用 DEB 包管理系统。

## 主要发行版

- **Debian** - 稳定、自由的社区发行版
- **Ubuntu** - 基于 Debian 的流行发行版
- **Linux Mint** - 基于 Ubuntu 的用户友好发行版
- **Deepin** - 中国开发的基于 Debian 的发行版

## 特点

- 使用 DEB 包管理
- 使用 apt/apt-get 作为包管理器
- 系统配置文件位于 /etc 目录
- 服务管理使用 systemd
