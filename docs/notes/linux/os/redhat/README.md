# 红帽系 Linux

红帽系 Linux 是基于 Red Hat 的 Linux 发行版，使用 RPM 包管理系统。

## 主要发行版

- **Red Hat Enterprise Linux (RHEL)** - 企业级商业发行版
- **CentOS** - RHEL 的社区克隆版
- **Fedora** - RHEL 的上游社区版
- **Rocky Linux** - CentOS 的继任者
- **AlmaLinux** - CentOS 的另一个继任者

## 特点

- 使用 RPM 包管理
- 使用 yum/dnf 作为包管理器
- 系统配置文件位于 /etc 目录
- 服务管理使用 systemd
