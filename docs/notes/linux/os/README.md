# Linux 操作系统

Linux 有许多不同的发行版，主要分为两大主流分支：红帽系和Debian系。

## 目录结构

- [红帽系](./redhat/)
- [Debian系](./debian/)

## 主要发行版

### 红帽系
- Red Hat Enterprise Linux (RHEL)
- CentOS
- Fedora
- Rocky Linux
- AlmaLinux

### Debian系
- Debian
- Ubuntu
- Linux Mint
- Deepin
